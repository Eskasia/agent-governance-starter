# Code of Conduct

## Our Pledge

We aim to keep this project useful, focused, and respectful for people contributing to agent governance tooling.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different experience levels and tool preferences.
- Keep discussion focused on evidence, reproducible behavior, and maintainable changes.
- Give and receive technical feedback in good faith.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without explicit permission.
- Repeated off-topic disruption.
- Misrepresenting project scope, adoption, or security guarantees.

## Enforcement

Maintainers may remove comments, close issues, or block participation when behavior makes the project unsafe or unproductive. Enforcement decisions should be proportionate to the behavior and documented when practical.

## Reporting

For security issues, use `SECURITY.md`. For conduct concerns, contact the maintainers through the repository issue or discussion channels when appropriate, and avoid sharing private information publicly.

This Code of Conduct is inspired by the Contributor Covenant and adapted for a small open-source project.
