# Release Handoff

## Trigger

Use before release, tester handoff, or cross-agent continuation.

## Steps

1. List changed surfaces and profile documents touched.
2. Report verification commands passed, skipped, and blocked.
3. Summarize remaining OPEN_LOOPS.md items.
4. Name the next owner and the exact next command or review step.
